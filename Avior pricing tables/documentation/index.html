HTML Page Markup

First off, make sure you have a solid code editor, for example VS Code and install the plugin Live Scss Compiler for compiling the scss.. You need to add the compiled main.css to the header of your HTML5 document. And that's it!

Example

                        <!DOCTYPE html>
                        <html>
                            <head>
                                <title></title>
                                <link rel="stylesheet" href="../main.css" />
                                
                            </head>
                            <body>

How to Add Table?

Hey! you can add table easily. so please follow these step and screenshot.

  1. At first you go to the source.
  2. Then open all-pricing-tables.html on editor (example: notepad++, VS Code etc)

  3. Suppose you want to a add basic pricing table on your project just add the following code in the body.

    Example

                <body>
                    ...

                        <div class="pricing-table" >
                        
                            <div class="price-column" >
                                <div class="header">
                                    <img class="column-img" src="assets/images/mountain.svg"/>
                                    <h2 class="column-heading">Basic</h2>
                                </div>
                                <div class="column-horizontal-gap"></div>
                                <div class="column-price">9<span>$</span></div>
                                <ul class="column-item-list">
                                    <li>Lorem ipsum dolor sit.</li>
                                    <li>Lorem ipsum dolor sit amet.</li>
                                    <li>Lorem, ipsum dolor.</li>
                                    <li>Lorem ipsum dolor sit.</li>
                                    <li>Lorem ipsum dolor sit amet.</li>
                                </ul>
                                <div class="btn-box">
                                    <a class="btn btn-go" href="#">BUY NOW</a>
                                </div>
                            </div>

                        </div>

                    ...
            </body>

    Options Uses

    Class Description
    pricing-table Base class for pricing tables
    rounded Changes the tables style from squre edges to rounded.
    highlighted Changes the tables style from normal to hightlighting gloss.
    striped Changes the table list style to striped.
    text-only Need to remove icon before using this.
    table-icons Works whlie using the icon in table lists.
    active Highlight the active column with primary color.
    table-red Changes the color theme of tables to red.
    table-green Changes the color theme of tables to green.
    table-blue Changes the color theme of tables to blue.
    table-dark Changes the color theme of tables to dark.
    table-yellow Changes the color theme of tables to yellow.
    table-purple Changes the color theme of tables to purple.

    Note: As same way you can add any table style to your project.

  4. Finally just save it and view in your browser. Now you are done.
