HTML Page Markup

First off, make sure you have a solid code editor, for example VS Code and install the plugin Live Scss Compiler for compiling the scss.. You need to add the compiled and preferably buttons.css, font-awesome.min.css or CDN to the header of your HTML5 document. And that's it!

Example

<!DOCTYPE html>
                        <html>
                            <head>
                                <title></title>
                                <link rel="stylesheet" href="stylesheets/buttons.css" />
                                <link rel="stylesheet" href="stylesheets/font-awesome.min.css" />
                            </head>
                            <body>
                                <a href="#1" class="fw-btn fw-basic fw-warning">BUTTON</a>
                                .
                                .
                                .
                            </body>
                        </html>

Note: You can make any element a button by using the classes .

How to Add Button?

Hey! you can add button easily. so please follow these step and screenshot.

  1. At first you showed go documentation folder
  2. Then open index.html on editor (example: notepad++, VS Code etc)

  3. Suppose you want to a add basic button on your project. So, please add a anchor element in the body.

    Example

    <body>
        <a href="#" class="fw-btn fw-basic fw-warning fw-large fw-round">BUTTON</a>
        ...
</body>

    Options Uses

    Class Description
    fw-basic Style of the button, related class (fw-3d, fw-modern etc).
    fw-warning This class use for color preset, related class (fw-info, fw-primary, fw-dark etc).
    fw-large Used for create large button, related class (fw-small, fw-medium)
    fw-round Used for create round button related class (fw-round, fw-pill)

    Note: As same way you can add any button style your project.

  4. Finally just save it and view your browser. Now you are done.
